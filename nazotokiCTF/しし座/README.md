# Solution
画像が与えられているので、何らかの文字を削除すると予想できる。  
プロローグを見てもそれらしきものはないのでルールを見る。  
意味不明な以下のルールがみられる。  
```
大切なルール
・燃えるゴミは捨ててください
・ペットボトル
```
これらを削除すればよいと考え画像から`モエルゴミ`と`ペットボトル`の文字を取り除く。
