# Solution
2019のwelcome問のフラグを探せばよい。  
「taskctf 2019 writeup」などとググると[このような記事](https://st98.github.io/diary/posts/2019-12-06-taskctf.html)が見つかり、`taskctf{let's_enj0y!}`であったとわかる。  
これがflagであった。  

## taskctf{let's_enj0y!}
